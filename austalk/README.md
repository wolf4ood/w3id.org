#  Austalk

This [W3ID](https://w3id.org) provides a persistent URI namespace for Austalk resources such as the profile, schema, and JSON-LD context.

These URIs are maintained by the [Language Data Commons of Australia](https://ldaca.edu.au), a project led by the University of Queensland.

## Schema/Vocabulary for Austalk resources
* https://w3id.org/austalk/schema
